This section contains the downloads and patches for the current version and all previous versions of JARVIS. It also contains packs that I have created that I feel not everyone would need or want, they are optional to download and are incorporated into your JARVIS using the 'load, command' command.

To properly update your JARVIS using the patches, first find your version number by calling the command 'version'. Then download every subsequent patch and install them in order of version. If you do this incorrectly or out of order then you may run into issues while using your program.
